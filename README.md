# WordPress Content Repository

This repository follows the WordPress content-directory layout:

```text
wp-content/
├── mu-plugins/                         # Must-use plugins shared by the site.
├── plugins/                            # Standard site plugins.
└── themes/
    └── nolan-young-theme-template-01/  # The classic WordPress theme and its tooling.
```

Run theme development commands from [`wp-content/themes/nolan-young-theme-template-01`](wp-content/themes/nolan-young-theme-template-01/README.md).
